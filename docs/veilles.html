Mes Veilles

Afin de mettre au point ces veilles, plusieurs outils ont été utilisé afin de récupérer les informations d'intérêts. Les extensions de navigateurs Peerltrees et Firefox pocket me permettent de mettre de côté des pages récupérées depuis internet afin de les retrouver plus tard.Aussi, la récupération de newsletters et flux RSS ont été établis depuis l'aggregateur Netvibes par l'intermédiaire d'un compte Twitter professionel où divers groupes ont été follow afin que netvibes récupére les tweets d'intérêts. Afin, sur un compte Discord pro, plusieurs groupes ont été rejoin afin de pouvoir pleinement profiter des effets collaboratifs de ce type de plateforme afin de recevoir et découvrir les ressources partagées par ce type de réseau.

Parmis ces comptes il est possible de citer: Backseat, CNIL, cyberveillance.gouv ou encore signal-arnaques pour la veille juridique, et des groupes comme GitHub Changelog, Gitlab, Laminas, Gartner ou IT-Connect. Enfin, d'autres sources dont les articles sont plus généraux, telles que Presse_Citron, ZDNet, FrenchWeb, Ars Technica, WIRED ou encore Bleeping Computers, ont été ajouté à l'aggregation de flux.

Une veille est un outil important dans le domaine de l'informatique qui évolue sans cesse. Ma veille technologique porte sur le framework Laminas. Quant à ma veille juridique, celle-ci porte sur la sécurité sur le web et les outils permettant de rester vigilant.
